# Get start
`npm i next`
`npm i react react-dom`

front/pages에 page.js 생성 (폴더 이름 변경 불가)

front/component에 component (폴더 이름 변경 가능)

### 기억해두기, props.chilren으로 하휘 요소 랜더링하는 방법

### 기억해두기, next는 react hot loader(nodemon같은 거)가 적용되어 있음
